Welcome to my new website! This is the first post in my "Notes" section, powered by **Astro Content Collections**.

### The Stack

The site is built using Astro, React, and Tailwind CSS. The interactve 3D point cloud on the home page is rendered using React Three Fiber, allowing for procedural point generation and cinematic rotation while maintaining 60FPS.
